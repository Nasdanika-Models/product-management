  • Load specification

A unit of utility for one or more personas — something that helps to achieve a goal, satisfies a need, or addresses a pain point.

One feature may be associated with multiple goals, needs, and pain points and vice versa. Features can be associated with personas directly for coarse-grained models.

A Feature is modeled as a subtype of Capability because a feature is itself a capability exposed to users.
