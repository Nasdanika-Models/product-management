  • Load specification

A concrete, observable step a persona performs in a workflow or scenario. Actions are contextual, sequential, and task-level.

They capture: - Task steps in a workflow - Interactions with tools or systems - Triggered events (what they do next) - User journey touchpoints

Examples: - ‘Uploads a resume file’ - ‘Filters candidates by skill score’

Actions help design what the user must do to succeed. They guide workflow and feature design.
